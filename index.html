  • * - универсальный селектор. Выбирает все.
  • h2 - выбирает все элементы указанного типа.
  • .class - выбирает элемент у которого есть атрибут class с указанным значением
  • #id - выбирает элемент у которого есть атрибут id с указанным значением
  • tag - применяется ко всем указанным элементам. Часто используется для задания цвета, фона, шрифта и др.
  • a:link, a:visited, a:hover, a:active, a:focus - выбирает элементы (обычно это поля формы или ссылки) в зависимости от их состояния.
  • [attr] - выбирает элементы, которые содержат указанный атрибут.
  • :checked - cтилизует выбранные radio / checkbox.
  • :focus - выбирает активный HTML элемент, тот который находится в фокусе. Обычно применяется к элементам формы.
